# Chapter 07

## Prerequisites

The following software is required to run the sample:

- Git
- Node.js
- npm
- Bash

## Running

Go to the current directory and run npm install in each subdirectory:

```sh
cd blue
npm install
cd ..

cd green
npm install
cd ..

cd red
npm install
cd ..

cd gateway
npm install
cd ..
```

Now start the application:

```sh
./run.sh
```
